# Bulk User Management System

This project implements a scalable backend system for managing users in bulk using **Node.js, Express, and MongoDB**.

It supports **bulk creation and bulk updates of users (5000+ records)** using efficient MongoDB operations like `insertMany()` and `bulkWrite()`.

Key features include:

* Schema validation and unique constraints
* Database indexing for performance optimization
* Bulk operations for large datasets
* Database export in **BSON and JSON** formats
* API testing with **Postman**

The system is designed to handle high-volume user data efficiently while maintaining data integrity and performance.
